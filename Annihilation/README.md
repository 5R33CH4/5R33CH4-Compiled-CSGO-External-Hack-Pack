## FUNCTIONS
**AIMBOT**<br>
[F6] - Enable Aimbot<br>
[F3] - BodyAim/HeadAim<br>
[F7] - FOV +0.2<br>
[F8] - FOV -0.2<br>
[F9] - Smooth +0.2<br>
[F10] - Smooth -0.2<br>
**ESP**<br>
[F11] - Enable GLOW ESP<br>
[F1] - Change the style<br>
**ADDITIONALLY**<br>
[V] - Enable / disable TriggerBot<br>
[ALT] - TriggerBot aiming during shooting<br>
[F2] - BunnyHop<br>
[INS] - Show the rank of players<br>
